Dataserv-Client Setup

Until enough old client are moved over to 2.0.0+ newer clients will have to use the url http://status.driveshare.org:5000. We strongly recommend everyone to move to the latest build so that development can move faster.

Note if you have previously built using version 1.3.0 keep in mind that:

  • You can only register 1 payout address per machine until multiple paths are available.
  • If you register the same address used with version 1.3.0 as the payout address and the same <PATH TO FOLDER> to store data the build process should be really fast.
  • The payout address must hold a balance of 10,000 SJCX for a Milestone #3 reward, you are free to test the software, for no rewards, if you feel like contributing to the project.
  • The server will register your address for payouts and provide another server-generated address which will come up in the stats page at: http://status.driveshare.org:5000/api/online

  • Windows

Download

Download the latest Storj Dataserv-client Windows release from GitHub.

Extract

Extarct the zip file to the folder where you wish to have it installed.

The dataserv-client will automatically update when new releases are made.

Install Client

$ sudo apt-get install python3-pip
$ sudo pip3 install dataserv-client
$ dataserv-client version

Update Client

$ sudo pip3 install dataserv-client --upgrade
$ dataserv-client version

Instructions

Farmer quickstart guide

Configure the client

Optionally set a cold storage payout address. You can set a counterparty wallet address here. All configuration must be done before starting the node. dataserv-client.exe --url=http://status.driveshare.org:5000 config --set_payout_address=<YOUR ADDRESS> Replace <YOUR ADDRESS> with your SJCX address. The <> are simply placeholders, delete them before pressing enter.

This will let you use a cold storage wallet or a counterparty wallet as the payout address instead of the automatically generated wallet. If you have no counterparty wallet and wish to use the generated wallet, please make a secure backup of the generated config.

Start farming

Optionally specifiy the path to store data path and available space. dataserv-client --url=http://status.driveshare.org:5000 --max_size=<YOUR SIZE HERE> farm

Make sure you choose an amount of storage that you have space for on your hard drive, i.e. 1TB for 1 terabyte, 5GB for 5 gigabytes, etc.

You will then see your command prompt continuously displaying: Pinging http://status.driveshare.org:5000 with address <YOUR ADDRESS> Once you are pinging you are done, keep it running.
